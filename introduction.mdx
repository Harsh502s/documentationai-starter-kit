## Key Features

Harshit Singh Documentation offers a robust set of features designed to streamline your documentation workflow. Our platform supports multiple documentation formats, real-time collaboration, and comprehensive search capabilities.

<Tabs>
  <Tab title="Basic" icon="file-text">
    Start with simple markdown pages and basic organization.
    
    <Steps>
      <Step title="Create Account" icon="user">
        Sign up for a free account to get started.
        ```bash
        curl -X POST https://api.harshitdocs.com/auth/register \
          -H "Content-Type: application/json" \
          -d '{"email":"user@example.com","password":"securepass123"}'
        ```
      </Step>
      <Step title="Create First Doc" icon="file">
        Use our API to create your first documentation page.
        ```javascript
const axios = require('axios');

async function createDoc() {
  try {
    const response = await axios.post('https://api.harshitdocs.com/docs', {
      title: 'My First Document',
      content: '# Welcome\nThis is my first doc.',
      projectId: 'proj-123'
    }, {
      headers: { 'Authorization': 'Bearer YOUR_TOKEN' }
    });
    console.log('Doc created:', response.data.id);
  } catch (error) {
    console.error('Error:', error.message);
  }
}

createDoc();
```
      </Step>
    </Steps>
  </Tab>
  <Tab title="Advanced" icon="settings">
    Leverage advanced features like versioning and webhooks.
    
    <ExpandableGroup>
      <Expandable title="Version Control" default-open="false">
        Track changes and maintain multiple versions of your documentation.
        
        <Callout kind="tip">Use semantic versioning for your docs to keep track of major and minor changes.</Callout>
      </Expandable>
      <Expandable title="Webhooks" default-open="false">
        Set up webhooks to notify external systems when docs are updated.
        
        ```javascript
// Example webhook payload
const webhookPayload = {
  event: 'doc.updated',
  docId: 'doc-456',
  changes: {
    title: 'Updated Title',
    lastModified: '2024-10-15T10:30:00Z'
  }
};

// Send to your endpoint
fetch('https://your-webhook-endpoint.com/notify', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify(webhookPayload)
});
```
      </Expandable>
    </ExpandableGroup>
  </Tab>
